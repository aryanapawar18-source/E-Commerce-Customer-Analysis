# E-Commerce-Customer-Analysis
🔷 Overview

This project focuses on analyzing e-commerce customer data to uncover insights related to sales performance, customer behavior, and business growth. The analysis is performed using Excel, MySQL, Power BI, and Tableau to deliver data-driven insights and interactive dashboards.

🔷 Objectives
Analyze customer purchasing behavior
Identify top-performing product categories
Measure key business KPIs
Understand customer demographics and segmentation
Provide actionable insights for decision-making

🔷 Tools & Technologies
Microsoft Excel – Data Cleaning, Pivot Tables, Dashboard Creation
MySQL – Data Querying, Joins, Aggregations
Power BI – Interactive Dashboards, DAX Measures
Tableau – Data Visualization & Storytelling

🔷 Dataset
E-commerce customer dataset containing:
Customer ID
Purchase Date
Product Category
Product Price
Quantity
Total Purchase Amount
Payment Method
Customer Age
Gender

🔷 Key KPIs
Total Revenue
Average Order Value (AOV)
Total Customers
Customer Churn Rate
Revenue by Product Category
Payment Method Distribution
Customer Age Distribution

🔷 Project Workflow
Data Collection & Import (CSV to MySQL)
Data Cleaning (Excel & SQL)

🔷 Dashboards

The project includes interactive dashboards built using:
Power BI
Tableau
Excel

These dashboards help visualize KPIs like revenue trends, customer segmentation, and sales performance.

🔷 Key Insights
High revenue generated from top product categories
Majority of customers belong to a specific age group
Certain payment methods are more frequently used
Customer churn is higher among low-frequency buyers
